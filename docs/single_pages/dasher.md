# Dasher

## Use

You use the dasher like a normal item with right-click and will get a boost in the direction you are looking at.
After using it the item has a cool down like enderpearls and is applied to all dashers you may have in your inventory.
So it is not useful having multiple dashers to have more or greater boosts.
The dasher also has infinit durability and can be destroyed by fire.
